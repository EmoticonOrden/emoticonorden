**Some rules for contributing to my projects.**
## Rules
1. If your pull request does not align with the project vision or plan, I may reject your pull request without providing a reason.
2. If you'd like to contribute code generated by AI to a project, please include 🤖 in the title or include information about the AI ​​in the description of your pull request.
3. Specify labels (in the issue itself) or tags of the type [Question], [Feature], etc. to correctly process the request.
4. More rules or clarifications will be added later.
